🎨 Графика и Персонажи
Все визуальные ассеты персонажей зафиксированы в файле models.png, расположенном в корневом каталоге проекта. Карта спрайтов содержит:

Главный Герой (Player): Кибер-самурай в конической неоновой шляпе, высокотехнологичном плаще и с имплантированным светящимся сердцем на броне.

Враг (Enemy): Корпоративный штурмовик в тяжелой броне со светодиодным визором и монолитным ростовым щитом, блокирующим проход.

Для интеграции этих моделей в сетку 25x25px используются свойства CSS image-rendering: pixelated; и фильтр drop-shadow, что позволяет сохранить оригинальную четкость пикселей из models.png без смазывания браузером.

🎮 Управление в игре
Игра считывает нажатия клавиш в реальном времени:

Стрелочки (Вверх, Вниз, Влево, Вправо) — Перемещение ронина по клеткам пола.

Пробел (Space) — Взмах катаной (Атака ближнего боя). Наносит урон врагу, если он находится в соседней клетке. Вызывает анимацию slash на игровом поле.

🚀 Быстрый запуск
Склонируйте или скачайте эту папку с проектом.

Откройте папку целиком в редакторе (например, в VS Code через File -> Open Folder). Это критически важно, чтобы у расширений вроде GitLens или Live Server не возникало ошибок абсолютных путей (Base path error).

Запустите локальный сервер, нажав кнопку Go Live в нижнем правом углу VS Code.

Игра автоматически откроется в вашем браузере по адресу http://127.0.0.1:5500/.
